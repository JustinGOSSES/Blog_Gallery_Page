Data Visualizations, Games, and Map Gallery

Name:

ISS Flyover

Tool:

APIs & Leaflet.js

Type:

Map

Description:

Map that shows the real-time position of the International Space Station above Earth on a two dimensional map. A marker is added every five seconds. If the map is left up long enough, the markers will make a complete lap around the earth roughly every 90 minutes.

 

Name:

Gource Viz of Making ISS Flyover

Language:

executes from terminal

Type:

Movie of Git Logs

Description:

Three dimensional visualization of github logs in movie form. This example uses the git logs from building the ISS Flyover Map. It represents authors changing files and directories via little avators that shoot bolts of light. Executes from terminal. You can changes options or just use defaults, nothing else required.

2048

Language:

Python

Type:

Interactive Game

Description:

2048 Numbers Game Another project built for the online Python Class I was taking, 2048 is a game in which you move numbers around a board. Like numbers merge in pairs. For example blocks of 2 and 2 becomes a single 4. The goal being to get a score of exactly 2048.
